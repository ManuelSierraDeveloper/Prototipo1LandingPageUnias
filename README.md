# Landing Page – Presentación para negocio de uñas

## ¿Qué es esta landing page?
Una landing page es una página web diseñada con un único objetivo: captar la atención del visitante y guiarlo hacia una acción concreta (reserva, llamada, suscripción, compra, etc.). Esta landing page está pensada específicamente para la presentación de la presencia online de un negocio de uñas (salón, técnico/a independiente o estudio de manicura).

## Objetivo principal
Presentar los servicios, mostrar el estilo y la calidad del trabajo, generar confianza y convertir visitantes en clientes mediante llamadas a la acción claras (reservar cita, llamar por teléfono, enviar mensaje por WhatsApp, o suscribirse a promociones).

## Público objetivo
- Personas interesadas en servicios de manicura y pedicura.
- Clientes que buscan inspiración (diseños, tendencias).
- Usuarios locales que necesitan reservar un servicio rápido.
